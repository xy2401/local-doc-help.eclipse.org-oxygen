dataModelElementFactory

Identifier:
org.eclipse.datatools.connectivity.sqm.core.dataModelElementFactory

Since:
0.7

Description:
Provides a factory for creating data model elements. The data model element factory, which must implement the org.eclipse.datatools.connectivity.sqm.core.definition.DataModelElementFactory interface, can be used to return an org.eclipse.emf.ecore.EObject instance of a given org.eclipse.emf.ecore.EClass

Configuration Markup:

<!ELEMENT extension (factory+)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED>


<!ELEMENT factory EMPTY>

<!ATTLIST factory

product CDATA #REQUIRED

version CDATA #REQUIRED

class   CDATA #REQUIRED>

  • product - Vendor name for the db definition.
  • version - Version name for the db definition.
  • class - The fully qualified name of a class that implements org.eclipse.datatools.connectivity.sqm.core.definition.DataModelElementFactory.

API Information:
A factory instance for a given database definition can be obtained using org.eclipse.datatools.connectivity.sqm.core.definition.DatabaseDefinition#getDataModelElementFactory()
