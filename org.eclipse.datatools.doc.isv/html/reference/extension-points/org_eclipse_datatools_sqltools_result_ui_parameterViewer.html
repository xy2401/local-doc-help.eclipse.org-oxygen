External Viewer for Parameter

Identifier:
org.eclipse.datatools.sqltools.result.ui.externalParameterViewer

Since:
[Enter the first release in which this extension point appears.]

Description:
Enables extensions to add alternate viewer for the parameters.

Configuration Markup:

<!ELEMENT extension (externalParameterViewer)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED>


<!ELEMENT externalParameterViewer EMPTY>

<!ATTLIST externalParameterViewer

vendor              CDATA #IMPLIED

class               CDATA #REQUIRED

default_viewer_name CDATA "Default Parameter Viewer"

id                  CDATA #REQUIRED>

  • vendor - Database vendor name
  • class - Extension must extend this class and override configureTableViewer().
  • default_viewer_name - The name of the parameter viewer to be used by the SQL Parameter Viewer if more than one exists.
  • id - The id of the viewer. This string will appear in the parameter viewer preference page.
