Provides supports for external sources to run and retrieve data to be displayed in Results view


  • vendor - vendor: String as returned by import org.eclipse.datatools.modelbase.sql.schema..Database.getVendor(). This value is used to identify the vendor for the purpose of external support for running and displaying sample content. If the value from the Database matches the value in the extension, then external support will be used.
  • class - The name of the class which implements IExternalRunQuery
