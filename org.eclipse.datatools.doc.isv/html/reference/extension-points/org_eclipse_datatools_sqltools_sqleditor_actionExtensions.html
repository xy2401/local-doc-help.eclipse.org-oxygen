SQL Editor Action Extensions

Identifier:
org.eclipse.datatools.sqltools.sqleditor.actionExtensions

Since:
[Enter the first release in which this extension point appears.]

Description:
This extension point allow tooling vendors to contribute additional context menus, toolbar items and workbench menus to the SQL Editor framework.

Configuration Markup:

<!ELEMENT extension (actionExtension+)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED>


<!ELEMENT actionExtension EMPTY>

<!ATTLIST actionExtension

editorId CDATA "all"

class    CDATA #REQUIRED>

  • editorId - could be "org.eclipse.datatools.sqltools.sqleditor.SQLEditor", "org.eclipse.datatools.sqltools.internal.sqlscrapbook.editor.SQLScrapbookEditor", "org.eclipse.datatools.sqltools.routineeditor.internal.RoutineEditor" or "all", which means all three types of editors.
