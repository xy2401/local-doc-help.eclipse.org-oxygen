DDL Parser

Identifier:
org.eclipse.datatools.connectivity.sqm.core.ddlParser

Since:
0.7

Description:
Provides the ddl parser for a given database definition. The ddl parser provided must implement the org.eclipse.datatools.connectivity.sqm.internal.core.rte.DDLParser interface.

Configuration Markup:

<!ELEMENT extension (parser+)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED>


<!ELEMENT parser EMPTY>

<!ATTLIST parser

product  CDATA #REQUIRED

class    CDATA #REQUIRED

version  CDATA #REQUIRED

priority CDATA #IMPLIED>

  • product - Vendor name for the db definition.
  • class - The fully qualified name of a class that implements org.eclipse.datatools.connectivity.sqm.internal.core.rte.DDLParser.
  • version - Version name for the db definition.
  • priority - The priority range is from 0 to 2147483647. The default is 0.

API Information:
An instance of the ddl parser can be obtained using org.eclipse.datatools.connectivity.sqm.core.definition.DatabaseDefinition#getDdlParser() method.
