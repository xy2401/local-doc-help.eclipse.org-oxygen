Database Contributions to SQL Development UI Tools

Identifier:
org.eclipse.datatools.sqltools.editor.ui.core.dbUIConfigurations

Since:
[Enter the first release in which this extension point appears.]

Description:
This extension point provides support for SQL development UI. Extensions register a dbUIConfiguration of the type org.eclipse.datatools.sqltools.ui.core.ISQLDevToolsUIConfiguration which will be responsible for creating vendor-specific configuration that will be used to customize the default behavior of the SQL Dev Tools UI framework.

Configuration Markup:

<!ELEMENT extension (dbUIConfiguration+)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED>


<!ELEMENT dbUIConfiguration EMPTY>

<!ATTLIST dbUIConfiguration

configurationClass CDATA #REQUIRED

product            CDATA #REQUIRED

version            CDATA #REQUIRED

description        CDATA #IMPLIED

default            (true | false) "false">

  • configurationClass - a fully qualified name of the Java class of the type org.eclipse.datatools.sqltools.ui.core.SQLDevToolsUIConfiguration which will be responsible for providing database specific classes.
  • product - the DBMS name
  • version - the DBMS version
  • description - optional description
  • default - Whether the configuration is the default one when no other configurations are registered against a database vendor id.

Examples: 


<extension
         point="org.eclipse.datatools.sqltools.editor.ui.core.dbUIConfigurations">
      <dbConfiguration
            product="Sybase Adaptive Server Enterprise"
            version="12.5.x"
            configurationClass="com.sybase.stf.dmp.ase.ASEUIConfiguration"
            >
      </dbConfiguration>
   </extension>

Contributes an ASE extenstion to the UI Configuration.
