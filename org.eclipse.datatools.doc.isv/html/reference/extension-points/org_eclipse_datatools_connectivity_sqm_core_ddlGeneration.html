DDL Generation

Identifier:
org.eclipse.datatools.connectivity.sqm.core.ddlGeneration

Since:
0.7

Description:
Provides an implementation of the org.eclipse.datatools.connectivity.sqm.core.rte.DDLGenerator and org.eclipse.datatools.connectivity.sqm.internal.core.rte.fe.DeltaDDLGenerator classes that can be used for generating DDLs for a given database definition.

Configuration Markup:

<!ELEMENT extension (generator+ , delta*)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED>


<!ELEMENT generator EMPTY>

<!ATTLIST generator

product  CDATA #REQUIRED

class    CDATA #REQUIRED

version  CDATA #REQUIRED

priority CDATA #IMPLIED>

  • product - Vendor name for the db definition.
  • class - The fully qualified name of a class that implements org.eclipse.datatools.connectivity.sqm.core.rte.DDLGenerator.
  • version - Version name for the db definition.
  • priority - The priority range is from 0 to 2147483647. The default is 0.

<!ELEMENT delta EMPTY>

<!ATTLIST delta

product  CDATA #REQUIRED

version  CDATA #REQUIRED

class    CDATA #REQUIRED

priority CDATA #IMPLIED>

  • product - Vendor name for the db definition.
  • version - Version name for the db definition.
  • class - The fully qualified name of a class that implements org.eclipse.datatools.connectivity.sqm.internal.core.rte.DeltaDDLGenerator.
  • priority - The priority range is from 0 to 2147483647. The default is 0.

API Information:
Use the methods org.eclipse.datatools.connectivity.sqm.core.definition.DatabaseDefinition#getDDLGenerator() and org.eclipse.datatools.connectivity.sqm.core.definition.DatabaseDefinition#getDeltaDDLGenerator() to obtain the generator instances for a given database definition.
