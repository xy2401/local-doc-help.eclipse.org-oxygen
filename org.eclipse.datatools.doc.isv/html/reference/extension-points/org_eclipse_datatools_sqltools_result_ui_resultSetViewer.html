External Viewer for ResultSets

Identifier:
org.eclipse.datatools.sqltools.result.ui.externalResultSetViewer

Since:
[Enter the first release in which this extension point appears.]

Description:
Enables extensions to add alternate viewer for the result sets

Configuration Markup:

<!ELEMENT extension (externalViewer)+>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED>


<!ELEMENT externalViewer EMPTY>

<!ATTLIST externalViewer

vendor              CDATA #IMPLIED

class               CDATA #REQUIRED

default_viewer_name CDATA "Default Result Set Viewer"

id                  CDATA #REQUIRED>

  • vendor - Database vendor name
  • class - Extension must extend this class and override configureTableViewer().
  • default_viewer_name - The name of the result set viewer to be used by the SQL Results Viewer if more than one exists.
  • id - The id of the viewer. This string will appear in the result set viewer preference page.
