Subversive unique features

  • Realizes the full support of Repository location layouts, recommended by Subversion®.
  • Provides the way to browse through not just project structure but revisions too directly in the repository location view.
  • Allows to freeze svn:externals while creating tags and branches.
  • There is a way to search automatically for Eclipse projects in the repository's specified location.
  • Supports group operations which simplifies management of the multiple projects in the same workspace.
  • It is extensible and has well defined API.
  • Each Subversive plug-in release warrants support for the last three Eclipse Platform releases (and all related update releases), so that you won't be forced to change your working environment when you need a new Subversive version installed.
